A simple web app that lets you play media from multiple platforms in one place. Just paste any media link and watch or listen without switching between different websites.

## What it does

MediaFlow takes URLs from popular platforms and embeds them in a clean, unified player. No more opening multiple tabs or apps - everything plays in one place.

## Supported Platforms

- **YouTube** - Videos, playlists, shorts
- **Spotify** - Tracks, albums, playlists
- **SoundCloud** - Tracks and playlists  
- **Vimeo** - Videos
- **Twitch** - Live streams and VODs
- **Dailymotion** - Videos

## How to use

1. Copy any media URL from the supported platforms
2. Paste it into the input field
3. Click Play or press Enter
4. Media plays in the embedded player
